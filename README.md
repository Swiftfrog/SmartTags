# SmartTags

> Emby Plugin

**智能标签管家**

## ✨ 核心功能：

- 智能原产地：基于 TMDB 深度数据，智能判断华语片（港/台/陆）与国际合拍片的主导归属。支持自定义显示格式（如：香港 (HK)）。

- 制片商/流媒体 (New!)：自动识别 Netflix, Disney+, HBO, A24, TVB, 邵氏, 银河映像等全球 50+ 顶级厂牌。

- 媒体信息：自动为您的 4K/HDR/Atmos/杜比视界 影片添加技术标签，无需联网，秒级处理。

- 实时监听：新入库影片自动打标，无需手动运行计划任务。

- 全量扫描：可以对所有媒体库实现`标签`，且缓存。

## 🛡️ 安全特性：

- 安全回滚：提供精准的“清理任务”，可基于算法反向移除由插件生成的标签，保护您的手动标签。

- 智能防护：清理任务执行时自动暂停实时监听，防止冲突。

# ⚠️ 注意事项：

- 首次使用建议先在设置中填入 TMDB API Key。

- 清理任务具有危险性，使用前请阅读说明。
